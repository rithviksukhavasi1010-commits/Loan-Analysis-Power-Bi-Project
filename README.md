# Loan Analysis Dashboard using Power BI

## Project Overview
This project is an end-to-end Loan Analysis Dashboard developed using Power BI, SQL Server, Power BI Service, and DAX.

The dashboard analyzes:
- Loan amounts
- Loan default trends
- Customer demographics
- Financial risk metrics
- Credit score behavior
- Year-over-Year (YOY) loan changes
- YTD loan analysis

The project also includes real-world BI workflow features such as:
- Dataflows
- Gateway configuration
- Scheduled Refresh
- Incremental Refresh
- SQL Server integration

# Project Workflow

1. Imported dataset into SQL Server
2. Created Dataflow using Power BI Service
3. Connected Power BI Desktop to Dataflow
4. Performed data cleaning and transformation
5. Created DAX measures and calculated columns
6. Built interactive dashboard pages
7. Configured Incremental Refresh
8. Published report to Power BI Service
9. Configured Scheduled Refresh using Gateway
